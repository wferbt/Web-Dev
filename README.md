# Marketa (Marketplace Web App)

## Description
Marketa is a web marketplace application where users can:
- view products
- create and manage their own listings
- add products to favorites

## Tech Stack
- Frontend: Angular
- Backend: Django (planned)
- Database: PostgreSQL / SQLite

## Features
- Authentication (JWT)
- Product CRUD
- Favorites system
